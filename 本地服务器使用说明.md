## 🎯 工作原理

```
你的浏览器
    ↓ (http://localhost:3000)
本地服务器 (Node.js)
    ↓ (代理请求)
https://jiekou.ai/api
    ↓ (返回游戏代码)
本地服务器
    ↓ (返回给浏览器)
你的浏览器
```

**关键点：**
- 浏览器只与本地服务器通信（同源，无 CORS 问题）
- 本地服务器代理请求到外部 API
- 所有代码都在本地运行

## ⚠️ 常见问题

### Q: 端口 3000 被占用？
**A:** 修改 `server.js` 第 7 行：
```javascript
const PORT = 3000;  // 改为 3001 或其他端口
```

### Q: 找不到 node 命令？
**A:** 
1. 确认已安装 Node.js
2. 下载地址：https://nodejs.org/
3. 安装后重启命令行

### Q: 生成游戏时报错？
**A:** 
1. 检查服务器控制台日志
2. 确认网络连接正常
3. 检查 API 密钥是否有效
4. 查看错误详情

### Q: 如何停止服务器？
**A:** 
- 在运行服务器的窗口按 `Ctrl+C`
- 或关闭命令行窗口

## 🎯 提示词技巧

### ✅ 好的提示词特征：

1. **清晰描述核心玩法**
   - "玩家控制飞船射击敌人"
   - "通过合并生物来进化"

2. **指定游戏类型和风格**
   - "太空射击游戏，像素风格"
   - "平台跳跃，卡通风格"

3. **包含具体功能**
   - "有 Boss 战、道具系统、得分排名"
   - "包含 50 个关卡和计时模式"

4. **描述游戏目标**
   - "消灭所有敌人，拯救公主"
   - "收集所有星星，到达终点"

### ❌ 避免模糊描述：

- "做一个好玩的游戏" → 太模糊
- "随便做个什么" → 没有方向
- "像某某游戏但更好" → 不够具体

## ⚠️ 常见问题

### Q: 生成失败，提示错误

**可能原因：**
1. API 密钥无效或余额不足
2. 网络连接问题
3. API 服务暂时不可用

**解决方案：**
- 检查网络连接
- 确认 API 密钥有效
- 稍后重试

### Q: 生成的游戏无法运行

**可能原因：**
1. AI 生成的代码有 bug
2. 浏览器兼容性问题

**解决方案：**
- 重新生成一次
- 尝试修改提示词
- 查看浏览器控制台（F12）的错误信息

### Q: 生成速度很慢

**说明：**
- 生成复杂游戏需要 10-30 秒
- 这是正常的，AI 需要时间生成完整的游戏代码
- 请耐心等待进度条完成

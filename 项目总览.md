## 📝 使用示例

### 1. 创建太空射击游戏

```
创建一个太空射击游戏，玩家控制飞船在星空中穿梭，
躲避陨石并消灭外星敌人。要有多种敌人类型、
Boss 战、道具系统和得分排名。像素风格。
```

### 2. 创建合并养成游戏

```
创建一个合并养成游戏，通过合并相同的生物来进化和成长。
从单细胞生物开始，逐步进化成传奇生物。
可爱卡通风格。
```

### 3. 创建平台跳跃游戏

```
创建一个平台跳跃游戏，玩家需要灵活躲避机关和敌人，
收集星星到达终点。要有双重跳跃和冲刺能力。
```

## 💡 提示词技巧

### ✅ 好的提示词
- 清晰描述核心玩法
- 指定游戏类型和风格
- 包含具体功能要求
- 说明游戏目标

### ❌ 避免模糊
- "做一个好玩的游戏" → 太模糊
- "随便做个什么" → 没方向
- "像某某游戏" → 不够具体

### 模板
```
创建一个 [类型] 游戏，玩家 [玩法]，目标是 [目标]。
包含系统：[系统 1]、[系统 2]。
美术风格：[风格]
难度：[简单/中等/困难]
```

## ⚠️ 注意事项

1. **API 密钥**: 请妥善保管，不要公开分享
2. **网络连接**: 需要网络才能生成游戏
3. **浏览器**: 推荐使用 Chrome/Edge 最新版
4. **存储限制**: 本地存储约 5-10MB 容量

---

## 🐛 故障排除

### 生成失败
1. 检查网络连接
2. 验证 API 密钥
3. 查看浏览器控制台（F12）

### 无法保存
1. 清除浏览器缓存
2. 检查存储空间
3. 尝试其他浏览器

### 游戏不运行
1. 重新生成
2. 查看控制台错误
3. 简化提示词

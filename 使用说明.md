## ✅ 解决方案

由于 CORS（跨域资源共享）限制，浏览器不允许直接从本地文件调用外部 API。

### 有两种解决方案：

---

## 方案 A：使用 CORS 插件（推荐，最简单）

### 步骤 1：安装 CORS 插件

**Chrome/Edge 浏览器：**
1. 访问 Chrome 网上应用店
2. 搜索 "Allow CORS: Access-Control-Allow-Origin"
3. 点击"添加到 Chrome"
4. 安装后启用插件

**快速安装链接：**
```
https://chrome.google.com/webstore/detail/allow-cors-access-control/lhobafahddgcelffkeicbagpnnjnonoe
```

**Firefox 浏览器：**
搜索 "CORS Everywhere" 并安装

### 步骤 2：启用插件
- 点击浏览器右上角的插件图标
- 确保插件已启用（通常显示为蓝色或绿色）

### 步骤 3：打开生成器
双击打开文件：
```
simple-generator.html
```

或在浏览器地址栏输入：
```
file:///C:/Users/jiangym/.copaw/ai-game-platform/simple-generator.html
```

### 步骤 4：开始生成游戏
1. 输入游戏描述
2. 点击"生成游戏"
3. 等待 10-30 秒
4. 预览和试玩！

## ⚠️ 常见问题

### Q: 点击"生成游戏"后没有反应？
**A:** 99% 是 CORS 问题！
- 请安装 "Allow CORS" 插件
- 确保插件已启用
- 刷新页面后重试

### Q: 安装插件后还是不行？
**A:** 
1. 确保插件已启用（图标应该是彩色的）
2. 刷新页面
3. 按 F12 打开控制台查看错误信息
4. 尝试重启浏览器

### Q: 生成的游戏无法运行？
**A:**
- 重新生成一次（AI 每次结果可能不同）
- 查看浏览器控制台的错误信息
- 尝试简化提示词

### Q: 生成速度慢？
**A:** 
- 正常现象，AI 生成需要 10-30 秒
- 复杂游戏需要更长时间
- 请耐心等待进度条完成

### 为什么需要 CORS 插件？
浏览器出于安全考虑，默认阻止从本地文件（file://）向外部域名发起请求。CORS 插件会修改请求头，绕过这个限制。

### 生成的游戏是什么格式？
- 单个 HTML 文件
- 包含所有 CSS 和 JavaScript
- 使用 Canvas API 渲染
- 可以直接在浏览器中运行

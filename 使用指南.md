## ⚠️ 常见问题

### Q: 生成失败，提示"API 请求失败"

**可能原因：**
1. API 密钥无效或余额不足
2. 网络连接问题
3. API 服务暂时不可用

**解决方案：**
- 检查网络连接
- 确认 API 密钥有效且有余额
- 稍后重试
- 查看浏览器控制台（F12）的详细错误信息

### Q: 生成的游戏无法运行/黑屏

**可能原因：**
1. AI 生成的代码有 bug
2. 浏览器兼容性问题
3. 游戏资源加载失败

**解决方案：**
- 按 F12 打开浏览器控制台查看错误
- 重新生成一次（AI 每次生成可能不同）
- 尝试修改提示词，简化要求
- 更换浏览器（推荐 Chrome）

### Q: 生成速度很慢

**说明：**
- 生成复杂游戏需要 10-30 秒，这是正常的
- AI 需要时间生成完整的游戏代码（通常 1000-3000 行）
- 请耐心等待进度条完成

### Q: 游戏太简单/太难

**解决方案：**
- 在提示词中明确说明难度
- 例如："简单难度，适合新手玩家"
- 或："高难度，挑战性十足"
- 重新生成时调整提示词

### Q: 可以保存游戏吗？

**当前状态：**
- 点击"发布"按钮可以保存到浏览器本地存储
- 刷新页面后游戏会丢失
- 未来版本将支持导出为 HTML 文件

**临时解决方案：**
- 截图或录屏保存
- 复制游戏代码（按 F12 查看）
- 收藏这个页面

## 🎮 游戏控制说明

生成的游戏通常支持以下控制方式：

### 键盘控制
- **方向键 / WASD**：移动
- **空格键**：跳跃/攻击
- **Shift**：冲刺/特殊技能
- **P 键**：暂停游戏
- **R 键**：重新开始

### 鼠标控制
- **鼠标移动**：控制角色/准星
- **左键点击**：攻击/放置
- **右键点击**：特殊动作
- **滚轮**：缩放/切换武器

具体控制方式会在游戏开始时显示说明。

## ⚠️ 重要提示

### 必须保持代理服务器运行

- ✅ 启动 `start-proxy.bat` 后**不要关闭窗口**
- ✅ 每次使用前都要启动代理服务器
- ✅ 使用 Ctrl+C 停止服务器

### 浏览器刷新

如果代理服务器已启动但仍报错：
1. 刷新页面（F5 或 Ctrl+R）
2. 清除浏览器缓存
3. 检查控制台（F12）

## 🐛 常见问题

### Q: 提示"端口被占用"

**解决**:
```bash
# 关闭其他占用 3000 端口的程序
# 或修改 proxy-server.js 中的 PORT 为其他值（如 3001）
```

### Q: 找不到 Node.js

**解决**:
1. 安装 Node.js: https://nodejs.org/
2. 重启命令行
3. 验证：`node --version`

### Q: 仍然有 CORS 错误

**检查**:
- [ ] 代理服务器是否运行
- [ ] 是否刷新了页面
- [ ] 请求 URL 是否为 `http://localhost:3000/proxy/...`

# GameAI - AI 生成游戏平台

一个类似 Astrocade 的 AI 游戏生成平台，让用户可以通过自然语言描述来创建游戏。

## 🎯 项目特点

- **AI 游戏生成**: 用户只需用自然语言描述游戏想法，AI 即可生成完整游戏
- **现代化 UI**: 深色主题、渐变效果、流畅动画
- **响应式设计**: 支持桌面和移动设备
- **游戏库**: 展示和浏览已创建的游戏
- **分类筛选**: 按类型、热度等筛选游戏

## 📁 项目结构

```
ai-game-platform/
├── index.html          # 首页
├── create.html         # 创建游戏页面
├── games.html          # 游戏库页面
├── about.html          # 关于页面
├── css/
│   ├── style.css       # 主样式文件
│   └── pages.css       # 页面特定样式
├── js/
│   ├── main.js         # 主脚本
│   ├── create.js       # 创建游戏逻辑
│   └── games.js        # 游戏库逻辑
└── images/
    ├── logo.svg        # Logo
    └── hero-game.svg   # 首页横幅图
```

## 🚀 快速开始

### 方法 1: 直接打开

直接在浏览器中打开 `index.html` 文件即可预览。

### 方法 2: 使用本地服务器

```bash
# 使用 Python
cd ai-game-platform
python -m http.server 8000

# 使用 Node.js (需要安装 http-server)
npx http-server -p 8000
```

然后在浏览器中访问 `http://localhost:8000`

## ✨ 核心功能

### 1. 首页 (index.html)
- 精美的 Hero 区域
- 热门游戏展示
- 游戏分类浏览
- 工作流程介绍

### 2. 创建游戏 (create.html)
- **灵感按钮**: 随机生成游戏创意
- **文本输入**: 描述你的游戏想法
- **高级选项**: 选择游戏类型、美术风格、难度
- **生成进度**: 实时显示 AI 生成进度
- **游戏预览**: 生成完成后预览和试玩

### 3. 游戏库 (games.html)
- 游戏网格展示
- 搜索功能
- 分类筛选
- 加载更多

### 4. 关于页面 (about.html)
- 平台介绍
- 核心功能展示
- 联系方式
- 统计数据

## 🎨 设计特色

- **深色主题**: 护眼且专业的深色配色方案
- **渐变效果**: 紫色系渐变，科技感十足
- **流畅动画**: 悬停效果、过渡动画提升用户体验
- **响应式布局**: 适配各种屏幕尺寸

## 🔧 技术栈

- **HTML5**: 语义化标签
- **CSS3**: Flexbox、Grid、动画、渐变
- **JavaScript**: 原生 JS，无框架依赖
- **Canvas**: 游戏预览渲染

## 🔍 可能的旧控件位置

### 位置 1：游戏代码中的 UI 显示

飞机大战游戏中可能有静态显示生命值的 UI：

**位置**: 游戏代码中的 `renderUI()` 函数

**代码示例**：
```javascript
// 游戏中显示生命值的 UI（心形图标）
for (let i = 0; i < player.maxHp; i++) {
  ctx.fillStyle = i < player.hp ? '#fb7185' : 'rgba(255,255,255,0.15)';
  // ... 绘制心形
}
```

**说明**：
- ✅ 这是游戏画面的一部分，**不应该删除**
- ✅ 它会随着参数调整而动态变化
- ✅ 调整滑块后，这个显示会实时更新

### 位置 2：参数面板中的重复控件

如果参数面板中显示了两个生命值控件：

**可能原因**：
1. 缓存问题 - 浏览器缓存了旧版本的 JS
2. 重复渲染 - 代码中调用了两次 `render()`

**解决方案**：

#### 方案 1：清除缓存（推荐）

```bash
1. 按 Ctrl + Shift + Delete
2. 选择"缓存的图片和文件"
3. 点击"清除数据"
4. 刷新页面（Ctrl + F5）
```

#### 方案 2：强制刷新

```bash
按 Ctrl + F5 强制刷新页面
```

#### 方案 3：清除 SessionStorage

打开浏览器控制台（F12），执行：
```javascript
sessionStorage.clear();
location.reload(true);
```

### 如果是代码重复渲染

**检查**: `js/game-analyzer.js` 中的 `render()` 函数

**确保**：
```javascript

function analyzeAndRender(code, gameId) {
    analyze(code, gameId);  // 分析

}

// ❌ 不要重复调用
analyze(code, gameId);
render('propertiesContent');
render('propertiesContent');  // 删除这行
```

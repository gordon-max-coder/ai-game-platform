## 🎯 使用示例

### 示例 1：生成太空射击游戏

**提示词**:
```
创建一个太空射击游戏，玩家控制飞船在星空中穿梭，
躲避陨石并消灭外星敌人。要有多种敌人类型、
Boss 战、道具系统（激光、导弹、护盾）和得分排名。
美术风格：像素风。
```

**步骤**:
1. 打开 `create.html`
2. 输入上述提示词
3. 选择类型：动作，风格：像素
4. 点击"生成游戏"
5. 等待 15-30 秒
6. 试玩并保存

### 示例 2：保存和导出游戏

**步骤**:
1. 生成游戏后点击"发布"
2. 输入游戏名称
3. 游戏保存到"我的游戏"
4. 在 `my-games.html` 查看
5. 点击"导出"下载 HTML 文件

### 示例 3：导入外部游戏

**步骤**:
1. 打开 `my-games.html`
2. 点击"导入游戏"
3. 选择 HTML 文件
4. 游戏导入成功
5. 可以游玩和管理

## ⚠️ 已知限制

### 技术限制
1. **存储限制**: 本地存储有容量限制
2. **跨域问题**: 某些浏览器可能限制 file:// 协议的 API 调用
3. **API 依赖**: 需要网络连接和有效的 API 密钥

### 功能限制
1. **游戏复杂度**: AI 生成的游戏相对简单
2. **资源限制**: 无法使用外部图片/音频（需 base64 编码）
3. **多人游戏**: 暂不支持多人在线功能

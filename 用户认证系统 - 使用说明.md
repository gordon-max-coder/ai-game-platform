## 🚀 快速开始

### 1. 访问登录/注册页面

打开浏览器访问：
```
http://localhost:3000/auth.html
```

### 2. 注册新账户

1. 点击"立即注册"
2. 填写信息：
   - 用户名：3-20 个字符
   - 邮箱：有效的邮箱地址
   - 密码：至少 6 位
   - 确认密码：与密码一致
3. 勾选"同意服务条款"
4. 点击"注册"

**注册成功后：**
- ✅ 自动登录
- ✅ 跳转到首页
- ✅ 侧边栏显示用户名

### 3. 登录账户

1. 访问 `auth.html`
2. 输入邮箱和密码
3. （可选）勾选"记住我"
4. 点击"登录"

**登录成功后：**
- ✅ 跳转到首页
- ✅ 侧边栏显示用户信息
- ✅ 会话保持 7 天

## 💾 数据存储说明

### IndexedDB 结构

**数据库名称：** `GameAI_DB`

**存储对象：**

1. **users** - 用户信息
```javascript
{
    id: "user_1234567890_abc",
    email: "user@example.com",
    username: "username",
    password: "加密后的密码",
    createdAt: "2025-03-06T12:00:00.000Z",
    updatedAt: "2025-03-06T12:00:00.000Z",
    gamesCreated: 0,
    avatar: null
}
```

2. **sessions** - 会话信息
```javascript
{
    userId: "user_1234567890_abc",
    loginTime: "2025-03-06T12:00:00.000Z",
    expiresAt: "2025-03-13T12:00:00.000Z"
}
```

### 安全性

- ✅ 密码加密存储（Base64 + Salt）
- ✅ 会话自动过期（7 天）
- ✅ 本地存储，不上传云端
- ⚠️ 生产环境应使用更强的加密算法

## 🔧 API 使用示例

### 注册
```javascript
const result = await Auth.register({
    username: 'myname',
    email: 'my@email.com',
    password: 'mypassword'
});

if (result.success) {
    console.log('注册成功:', result.user);
}
```

### 登录
```javascript
const result = await Auth.login('my@email.com', 'mypassword');

if (result.success) {
    console.log('登录成功:', result.user);
}
```

### 登出
```javascript
Auth.logout();
```

### 获取当前用户
```javascript
const user = Auth.getCurrentUser();
if (user) {
    console.log('当前用户:', user.username);
}
```

### 检查登录状态
```javascript
if (Auth.isLoggedIn()) {
    console.log('已登录');
} else {
    console.log('未登录');
}
```

### 更新个人信息
```javascript
await Auth.updateProfile({
    username: 'newname'
});
```

### 修改密码
```javascript
await Auth.changePassword('oldpass', 'newpass');
```

## 🔒 安全建议

### 当前实现
- ✅ 密码加密存储
- ✅ 会话过期机制
- ✅ 输入验证
- ⚠️ 本地存储（生产环境需改进）

### 生产环境建议
1. **使用 HTTPS** - 加密传输
2. **强密码策略** - 至少 8 位，包含大小写和数字

4. **密码重置** - 邮箱验证重置
5. **账户锁定** - 多次失败后锁定
6. **会话管理** - 更短的过期时间

## 🐛 常见问题

### Q: 刷新页面后登录状态丢失？
**A:** 检查：
1. 是否加载了 `user-state.js`
2. 浏览器是否允许 IndexedDB
3. 会话是否过期（7 天）

### Q: 注册时提示"邮箱已被注册"？
**A:** 该邮箱已存在，尝试：
1. 使用其他邮箱
2. 直接登录
3. 清除 IndexedDB 数据后重试

### Q: 如何清除所有用户数据？
**A:** 打开浏览器开发者工具：
```javascript
// 清除 IndexedDB
indexedDB.deleteDatabase('GameAI_DB');

// 清除 localStorage
localStorage.clear();

// 刷新页面
location.reload();
```

### Q: 忘记密码怎么办？
**A:** 当前版本不支持密码重置，可以：
1. 清除 IndexedDB 数据
2. 重新注册

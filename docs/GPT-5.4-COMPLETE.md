## 🎮 使用方法

### 方法 1: 前端选择（推荐）
1. 打开 `http://localhost:3000/create.html`
2. 选择模型：**GPT-5.4 (最强 GPT🚀)**
3. 输入游戏描述，点击"创建"

### 方法 2: 修改默认配置
编辑 `.env`:
```bash
MODEL=gpt-5.4
```
重启服务器。

### 方法 3: API 切换模态框
1. 点击 API 状态栏
2. 选择 GPT-5.4
3. 自动保存

## ⚠️ 重要说明

### 参数限制
GPT-5.4 **没有特殊参数限制**，可以正常使用所有 OpenAI 标准参数：
- ✅ temperature (0-2)
- ✅ top_p (0-1)
- ✅ presence_penalty (-2 to 2)
- ✅ frequency_penalty (-2 to 2)
- ✅ max_tokens

> 对比：`gpt-5-mini` 有温度固定等特殊限制

### 成本优化建议
虽然 GPT-5.4 质量最高，但建议混合使用：


## 🔍 故障排查

### 常见问题

#### 1. 400 错误
**原因:** JSON 格式错误  
**解决:** 检查请求数据转义

#### 2. 401 错误  
**原因:** API Key 无效  
**解决:** 检查 `.env` 的 `API_KEY`

#### 3. 模型不在列表中
**原因:** 浏览器缓存  
**解决:** 强制刷新 (Ctrl+F5) 或清除缓存

#### 4. 生成黑屏
**原因:** Markdown 未提取  
**解决:** 检查 `server.js` 第 108 行 HTML 提取逻辑

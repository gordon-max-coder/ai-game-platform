# 🎮 游戏编辑器快速测试指南

> **版本**: v32  
> **测试时间**: 5 分钟

## 🚀 快速测试步骤

### 1️⃣ 启动服务器

```bash
# 方法 1: 使用批处理文件
启动服务器.bat

# 方法 2: 手动启动
npm start
```

确保服务器运行在 `http://localhost:3000`

---

### 2️⃣ 打开创作页面

浏览器访问：`http://localhost:3000/create.html`

---

### 3️⃣ 创建测试游戏

**在左侧输入框输入**:
```
创建一个贪食蛇游戏，红色蛇身，绿色食物
```

**点击**: `🚀 创建` 按钮

**等待**: AI 生成游戏（约 3-5 秒）

---

### 4️⃣ 打开编辑器

**点击**: 预览区上方的 `🎮 编辑器` 按钮

**应该看到**:
- ✅ 编辑器模态框打开（占据 90% 屏幕）
- ✅ 左侧显示组件树（Main、Player 等）
- ✅ 中间显示游戏预览 iframe
- ✅ 右侧显示属性面板
- ✅ 底部显示代码编辑器

---

### 5️⃣ 测试组件树

**点击**: 左侧组件树中的 "Player" 对象

**应该看到**:
- ✅ 组件树项高亮显示（蓝色左边框）
- ✅ 右侧属性面板显示 Player 的属性
- ✅ 属性包括：位置、速度、颜色、大小等

---

### 6️⃣ 测试属性编辑

#### 测试数值属性（速度）
1. 找到 "speed" 属性
2. 拖动滑块从 300 → 500
3. 旁边的数字输入框应该同步变为 500
4. 点击 "▶️ 运行" 按钮
5. 观察游戏中蛇的速度是否变快

#### 测试颜色属性
1. 找到 "color" 属性
2. 点击颜色选择器
3. 选择蓝色 (#0000ff)
4. 文本输入框应该同步显示 #0000ff
5. 点击 "▶️ 运行" 观察蛇的颜色是否变蓝

#### 测试位置属性
1. 找到 "position" 属性
2. 修改 X 值为 200
3. 修改 Y 值为 350
4. 点击 "▶️ 运行" 观察蛇的初始位置

---

### 7️⃣ 测试运行控制

**点击**: `▶️ 运行` 按钮
- ✅ 按钮变灰（禁用状态）
- ✅ "⏹️ 停止"按钮变为可用
- ✅ 游戏在 iframe 中运行

**点击**: `⏹️ 停止` 按钮
- ✅ "▶️ 运行"按钮恢复可用
- ✅ "⏹️ 停止"按钮变灰
- ✅ 游戏停止运行

---

### 8️⃣ 测试代码编辑

1. 滚动到底部代码编辑器
2. 在代码中找到一个数字（如速度值）
3. 直接修改它（如 `speed = 300` → `speed = 600`）
4. 点击 `✅ 应用` 按钮
5. 应该看到提示 "✅ 代码已应用到预览"
6. 点击 "▶️ 运行" 验证修改生效

---

### 9️⃣ 测试保存功能

**点击**: `💾 保存` 按钮

**应该看到**:
- ✅ 提示 "✅ 游戏已保存"
- ✅ 控制台显示 "💾 编辑器保存游戏代码"
- ✅ 返回创作页面，游戏代码已更新

---

### 🔟 测试导出功能

**点击**: `📦 导出` 按钮

**应该看到**:
- ✅ 浏览器下载一个 HTML 文件
- ✅ 文件名类似 "game.html"
- ✅ 双击打开可以正常运行游戏

## 🐛 常见问题排查

### 问题 1: 编辑器打不开
**检查**:
1. 浏览器控制台是否有错误
2. `game-editor.js` 是否正确加载
3. `create.html` 中是否添加了编辑器按钮

**解决**:
```bash
# 强制刷新浏览器缓存
Ctrl + F5 (Windows)
Cmd + Shift + R (Mac)
```

### 问题 2: 组件树为空
**原因**: 游戏对象解析失败

**检查**:
1. 控制台是否有 "解析游戏对象" 日志
2. `state.gameObjects` 数组是否有内容

**调试**:
```javascript
// 在浏览器控制台运行
console.log(GameEditor.getState());
```

### 问题 3: 属性修改不生效
**原因**: 游戏代码不支持 postMessage

**解决**:
- 当前版本使用硬编码示例对象
- v33 将实现智能解析和双向通信

### 问题 4: 样式显示异常
**检查**:
1. `game-editor.css` 是否正确加载
2. 浏览器是否缓存了旧 CSS

**解决**:
```bash
# 清除缓存并刷新
Ctrl + Shift + Delete → 清除缓存
```

## 🚀 快速使用

### 方法 1: 前端选择（推荐）

1. 打开创作页面：`http://localhost:3000/create.html`
2. 在模型选择下拉框中选择 **GPT-5.4 (最强 GPT🚀)**
3. 输入游戏描述，点击"创建"按钮

### 方法 2: 修改 `.env` 配置

编辑 `.env` 文件：
```bash
MODEL=gpt-5.4
```

然后重启服务器：
```bash
🔄 重启服务器.bat
```

### 方法 3: API 切换模态框

1. 点击创作页面的 API 状态栏
2. 在模型列表中选择 **GPT-5.4**
3. 自动保存配置

## ⚠️ 注意事项

### 1. 参数限制
GPT-5.4 **没有**特殊参数限制，可以正常使用：
- ✅ temperature (0-2)
- ✅ top_p
- ✅ presence_penalty
- ✅ frequency_penalty
- ✅ max_tokens

### 2. 成本优化
虽然 GPT-5.4 质量最高，但成本也较高：
- **日常开发:** 建议使用 `gemini-2.5-flash` ($0.003/次)
- **关键项目:** 使用 `gpt-5.4` 或 `claude-opus-4-6`
- **混合策略:** 90% Gemini + 10% GPT-5.4

### 3. 响应时间
- **简单请求:** ~1-2 秒
- **复杂游戏:** ~5-10 秒
- **超时设置:** 300 秒（5 分钟）

## 🔍 故障排查

### 问题 1: 400 错误
**原因:** JSON 格式错误  
**解决:** 检查请求数据，确保正确转义

### 问题 2: 401 错误
**原因:** API Key 无效  
**解决:** 检查 `.env` 中的 `API_KEY` 配置

### 问题 3: 超时
**原因:** 游戏代码太长  
**解决:** 增加 `max_tokens` 或简化需求

### 问题 4: 黑屏
**原因:** Markdown 代码块未提取  
**解决:** 检查 `server.js` 的 HTML 提取逻辑

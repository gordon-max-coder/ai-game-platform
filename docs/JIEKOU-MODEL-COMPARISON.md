## 💡 实用建议

### 如何选择？

**问自己 3 个问题**:

1. **预算多少？**
   - 极少 → `gemini-2.0-flash-lite`
   - 有限 → `gemini-2.5-flash`
   - 充足 → `claude-sonnet-4-6`

2. **质量要求？**
   - 能跑就行 → `gemini-2.0-flash-lite`
   - 还不错 → `gemini-2.5-flash`
   - 生产级 → `claude-sonnet-4-6`

3. **项目类型？**
   - 学习/原型 → `gemini-2.0-flash-lite`
   - 个人项目 → `gemini-2.5-flash`
   - 商业项目 → `claude-sonnet-4-6`

---

### 成本优化技巧

1. **分步生成**
   - 先用便宜模型生成框架
   - 再用高质量模型优化关键部分

2. **多轮迭代**
   - 便宜模型快速迭代创意
   - 满意后用高质量模型定稿

3. **混合使用**
   - 90% 场景用 Gemini
   - 10% 关键场景用 Claude

4. **Prompt 优化**
   - 详细 Prompt 减少迭代次数
   - 一次生成正确比多次便宜

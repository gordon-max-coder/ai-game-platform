## 🎯 使用场景

### ✅ 推荐使用

1. **快速原型开发**
   - 快速验证创意
   - 多次迭代测试
   - 成本敏感项目

2. **简单游戏**
   - 贪食蛇、打砖块等经典游戏
   - 逻辑简单的休闲游戏
   - 2D 小游戏

3. **学习测试**
   - 学习 AI 生成游戏
   - 测试不同提示词效果
   - 大量实验性生成

4. **批量生成**
   - 需要生成多个游戏
   - A/B 测试不同版本
   - 成本优先的项目

### ⚠️ 不推荐

1. **复杂游戏**
   - 需要复杂逻辑
   - 多系统交互
   - 高级 AI 行为

2. **高质量需求**
   - 商业项目
   - 对外展示
   - 用户体验要求高

3. **创新玩法**
   - 需要创造性思维
   - 复杂游戏设计
   - 独特机制实现

## 🚀 如何使用

### 方法 1: 前端选择器

```
1. 打开 create.html
2. 点击模型选择下拉框
3. 选择 "MiMo V2 Flash ⚡ (超快速🚀)"
4. 开始使用
```

### 方法 2: API 切换模态框

```
1. 点击 API 状态栏
2. 选择 "jiekou.ai"
3. 点击 "MiMo V2 Flash ⚡"
4. 点击 "✅ 确认切换"
```

### 方法 3: 修改 .env

```env
API_PROVIDER=jiekou
MODEL=xiaomimimo-mimo-v2-flash
```

然后重启服务器。

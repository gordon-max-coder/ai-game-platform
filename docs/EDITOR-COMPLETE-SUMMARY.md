## 🔧 技术实现

### 架构设计

```javascript
GameEditor 模块
├── state (状态管理)
│   ├── currentGame      // 当前游戏代码
│   ├── gameObjects      // 游戏对象列表
│   ├── selectedObject   // 选中的对象
│   ├── isPlaying        // 运行状态
│   └── autoSave         // 自动保存开关
│
├── elements (DOM 缓存)
│   ├── componentTree    // 组件树容器
│   ├── propertyPanel    // 属性面板
│   ├── canvasPreview    // 画布预览
│   ├── codeEditor       // 代码编辑器
│   └── buttons...       // 各种按钮
│
├── 核心函数
│   ├── init()           // 初始化
│   ├── parseGameObjects()  // 解析对象
│   ├── renderComponentTree()  // 渲染组件树
│   ├── renderPropertyPanel()  // 渲染属性面板
│   ├── updateProperty() // 更新属性
│   ├── runGame()        // 运行游戏
│   ├── stopGame()       // 停止游戏
│   ├── saveGame()       // 保存游戏
│   └── exportGame()     // 导出游戏
│
└── 工具函数
    ├── getIconForType() // 获取类型图标
    ├── capitalize()     // 首字母大写
    └── debounceSave()   // 防抖保存
```

### 关键技术点

#### 1. 防抖自动保存
```javascript
let saveTimer = null;
function debounceSave() {
    if (saveTimer) clearTimeout(saveTimer);
    saveTimer = setTimeout(saveGame, 1000);
}
```

#### 2. 属性类型识别
```javascript
if (typeof value === 'object') {
    // 向量属性 - 渲染 X/Y 输入框
} else if (typeof value === 'number') {
    // 数值属性 - 渲染滑块 + 数字输入
} else if (value.startsWith('#')) {
    // 颜色属性 - 渲染选择器 + 文本输入
}
```

#### 3. 实时属性同步
```javascript
iframe.contentWindow.postMessage({
    type: 'UPDATE_PROPERTY',
    objectId: objectId,
    property: property,
    value: value
}, '*');
```

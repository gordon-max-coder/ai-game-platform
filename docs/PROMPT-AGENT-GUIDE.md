**版本**: v1.0  
**创建时间**: 2025-01-15  
**状态**: ✅ 已集成

---

## 📋 功能概述

提示词智能体通过多轮对话引导用户完善游戏需求，显著提升 AI 生成游戏的质量。

### 工作流程

```
用户输入简单需求
    ↓
智能体检测需求简单
    ↓
启动引导流程（3-4 个问题）
    ↓
用户选择选项
    ↓
生成完整提示词
    ↓
发送给 API 生成游戏
```

---

## 🎯 核心功能

### 1. 游戏类型识别

支持 6 种游戏类型自动识别：

| 游戏类型 | 关键词 | 问题数 |
|---------|--------|-------|
| 🐍 贪食蛇 | 贪食蛇、蛇蛇、snake | 4 个 |
| 🧱 打砖块 | 打砖块、砖块、breakout | 4 个 |
| 🚀 射击游戏 | 射击、飞机、太空、飞机大战 | 4 个 |
| 🏃 平台跳跃 | 跳跃、平台、冒险、马里奥 | 4 个 |
| 🧩 益智游戏 | 拼图、消除、解谜、2048 | 3 个 |
| 🏃 跑酷游戏 | 跑酷、奔跑、地铁 | 4 个 |

### 2. 智能引导触发

**触发条件**（满足任一）：
- 提示词长度 < 15 字
- 细节描述 < 2 个（颜色、速度、难度等）

**不触发情况**：
- 包含"简单"、"基础"、"基本"等词
- 已有多个细节描述

### 3. 问题类型

#### 单选题
```
🎨 你希望游戏是什么视觉风格？
🌈 鲜艳多彩  ⚪ 极简风格  👾 像素复古  💡 霓虹灯
```

#### 滑块题
```
🐍 蛇的移动速度？
[====●=====]  🚶 正常
🐌 很慢 ← → ⚡ 飞快
```

#### 多选题
```
✨ 需要哪些额外功能？（可多选）
🚧 障碍物  💊 道具系统  🤖 AI 对手  ✨ 粒子特效
```

## 🔧 技术实现

### 文件结构

```
js/
├── prompt-agent.js      # 智能体核心逻辑（22KB）
├── create-new.js        # 集成智能体（v12）
css/
├── create-layout.css    # 智能体样式
create.html              # 智能体 UI（v12）
```

### 核心类

```javascript
class PromptAgent {
    // 游戏类型配置
    gameTypes = { snake, breakout, shooter, ... }
    
    // 状态管理
    state = {
        isActive: false,
        gameType: null,
        step: 0,
        answers: {},
        basePrompt: ''
    }
    
    // 主要方法
    detectGameType(prompt)      // 识别游戏类型
    needsGuidance(prompt)       // 判断是否需要引导
    startGuidance(prompt)       // 启动引导
    getNextQuestion()           // 获取下一题
    recordAnswer(id, answer)    // 记录答案
    generateFinalPrompt()       // 生成最终提示词
}
```

### 提示词生成

智能体根据用户选择生成详细提示词：

**示例输出**：
```
创建一个贪食蛇游戏。蛇的移动速度为 7 像素/帧。
初始长度为 5，每吃一个食物增长 1 节。
地图中随机生成障碍物。包含加速、减速、穿墙等道具。
吃食物和死亡时有粒子爆炸效果。

代码结构清晰，使用 IIFE 避免全局变量污染。
包含完整的游戏循环（开始、运行、结束）。
有分数显示和游戏结束界面。
支持键盘和触摸控制。
添加粒子效果和动画增强视觉体验。
使用鲜艳的渐变色彩，视觉效果丰富。
```

## 📝 最佳实践

### ✅ 推荐

1. **简短输入触发引导** - 用户输入<15 字时自动引导
2. **问题数量 3-4 个** - 避免用户疲劳
3. **图标 + 文字** - 提升可读性
4. **进度条显示** - 让用户知道还剩几题
5. **支持跳过** - 高级用户可直接跳过

### ❌ 避免

1. **问题太多** - 超过 5 题用户会烦躁
2. **专业术语** - 使用通俗易懂的语言
3. **强制引导** - 始终提供跳过选项
4. **复杂选项** - 每个问题选项不超过 5 个

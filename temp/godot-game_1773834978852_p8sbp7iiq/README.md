```
game_1773834978852_p_godot/
├── project.godot      # Godot 项目配置
├── scenes/

├── scripts/
```

## 🚀 如何运行
1. 用 Godot 4.2+ 打开此项目
2. 点击 ▶️ 运行按钮
3. 游戏将在 360x640 窗口中运行
